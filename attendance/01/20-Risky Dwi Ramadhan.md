# Rangkuman Pertemuan 1

1. Membahas materi apa saja yang akan didapat dalam 17 minggu kedepan
2. Membahas tentang pengenalan dengan Mobile Programming seperti:
   a. Macam-macam tool yang bisa digunakan seperti Android Studio, Flutter, dll
   b. Componen dalam Android Studio yaitu:
      - Activity : layar utama aplikasi.
      - Broadcast Receiver : untuk menerima dan mengirim sesuatu ke/dari aplikasi lain (info battery, notif telpon)
      - Services : background proses (memainkan musik di background, alarm, dll)
      - Conten Provider : manage data dari berbagai sumber
   c. Tahapan dalam membuat dan mendevelop aplikasi mobile diantaranya :
      - UI Desain menggunakan xml code
      - Application Development menggunakan Java/Kotlin
      - Building menggunakan DSL Gradle
      - Testing aplikasi
      - Terakhir melakukan proses Development
